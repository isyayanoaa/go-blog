# go-blog

个人博客项目，前后端分离架构。支持 Markdown 文章、朋友圈动态、在线运行 Go 代码。

---

## 技术栈

| 层级 | 技术 |
|------|------|
| 后端 | Go + Gin 框架 |
| 数据库 | PostgreSQL |
| 缓存 / Session | Redis |
| 前端 | Vue 3 + Vite |
| 部署 | 云服务器 + Nginx 反向代理 |

---

## 用户角色

| 角色 | 说明 |
|------|------|
| 游客 | 未登录，可浏览所有公开内容 |
| 普通用户 | GitHub OAuth 登录后，可评论、点赞 |
| 管理员 | GitHub OAuth 登录，系统识别 admin ID，可发布/编辑/删除所有内容 |

**界面规则**：所有人看到同一套干净界面。管理员登录后，相关位置悄悄出现「编辑」「发布」「删除」按钮，普通用户和游客不可见。

---

## 功能模块

### 1. 博客文章

- 文章列表：按时间倒序展示所有已发布文章
- 文章详情：Markdown 渲染，代码语法高亮
- 分类（Category）：每篇文章归属一个分类
- 标签（Tags）：每篇文章可打多个标签
- 归档（Archive）：按年/月归档展示
- 点赞：登录用户可点赞文章
- 评论：登录用户可评论，支持回复（一级回复）
- 管理员操作（登录后可见）：
  - 写新文章（Markdown 编辑器，支持图片上传）
  - 编辑已有文章
  - 删除文章
  - 保存草稿（不对外公开）

### 2. 朋友圈动态

- 动态列表：瀑布流展示，支持图片 / 视频
- 点赞：登录用户可点赞
- 评论：登录用户可评论，支持回复（一级回复）
- 管理员操作（登录后可见）：
  - 发布新动态（文字 + 多图 / 视频）
  - 删除动态

### 3. 可运行 Go 代码

- 文章中可嵌入可运行 Go 代码块
- 代码块右上角显示「运行」按钮
- **仅管理员可点击运行**，其他人只能查看代码
- 调用 Go Playground API 执行代码，输出结果展示在代码块下方

### 4. 评论系统

- 评论数据存储在自有 PostgreSQL 数据库
- 必须 GitHub 登录才能评论
- 支持回复某条评论（一级回复，不无限嵌套）
- 管理员可删除任意评论

### 5. 登录系统

- 仅支持 GitHub OAuth 登录
- 登录后 Session 存储于 Redis
- 管理员判断：GitHub 用户 ID 与服务端配置中的 admin ID 比对
- 无独立注册流程，GitHub 账号即身份

## 页面路由

```
/                    首页（最新文章 + 最新动态摘要）
/posts               文章列表
/posts/:id           文章详情（含评论、可运行代码）
/moments             朋友圈动态
/categories          分类列表
/categories/:name    某分类下的文章
/tags                标签云
/tags/:name          某标签下的文章
/archives            按年/月归档
/about               关于我
```

---

## API 规范

- 所有后端接口统一前缀：`/api/v1/`
- 数据格式：JSON
- 前后端完全分离，后端只提供 REST API

---

## 其他说明

- 移动端适配（响应式布局）
- 图片 / 视频上传存储于服务器本地，后期可迁移至对象存储（OSS / Cloudflare R2）
- 评论、点赞数据均存自有数据库，不依赖第三方服务
